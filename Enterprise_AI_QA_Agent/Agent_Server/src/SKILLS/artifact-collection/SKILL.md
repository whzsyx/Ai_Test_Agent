# Artifact Collection

Use this skill whenever test execution produces evidence.

Collect and label:
- Screenshots
- DOM snapshots
- HTML captures
- API request/response records
- Console and network logs
- App maps and summaries
- Final reports

Prefer stable absolute paths and include artifact type, label, and purpose.
