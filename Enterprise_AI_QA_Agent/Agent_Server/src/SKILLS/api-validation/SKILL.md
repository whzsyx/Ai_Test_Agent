# API Validation

Use this skill when testing HTTP APIs, service contracts, or integration payloads.

Cover:
- Endpoint and method
- Required headers and authentication
- Request payload
- Expected status code
- Response schema and key fields
- Negative cases
- Idempotency and error handling
