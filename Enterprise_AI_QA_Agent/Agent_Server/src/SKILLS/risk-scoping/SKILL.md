# Risk Scoping

Use this skill before planning execution when the request could have multiple risk areas.

Prioritize:
- User-impacting failures
- Data loss, incorrect state, or irreversible actions
- Authentication, authorization, and privacy concerns
- Browser compatibility and UI regressions
- API contract and integration risks
- Performance bottlenecks and unstable dependencies
