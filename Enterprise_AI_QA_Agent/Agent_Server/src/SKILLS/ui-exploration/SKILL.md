# UI Exploration

Use this skill for UI Access Bootstrap, Page Exploration, Page Modeling, and App Map generation.

Prefer registered UI tools:
- `ui-page-explorer` for page discovery and app_map.json
- `browser-control` for explicit snapshot, screenshot, eval, and CLI-shaped actions
- `browser-automation` for ordered click, fill, type, press, wait, and screenshot flows

Always return artifact paths for evidence when available.
